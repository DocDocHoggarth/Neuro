Learning Targets

  • Distinguish canal, otolith, nerve, and nuclear functions.
  • Sequence vestibular mechanotransduction events accurately.
  • Sort stimuli by rotation, translation, and multisensory balance demands.
  • Explain push-pull coding and sensory reweighting in equilibrium.

How To Run It

  • Project one challenge at a time.
  • Let pairs make predictions before checking.
  • Use reset buttons for fast replay with a new group.
  • Pause after each check for a short debrief.
