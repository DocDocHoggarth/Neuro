# Neuro

This repository contains interactive learning tools designed to help students build their understanding of the nervous system. The activities focus on neuroanatomy and neuroscience concepts through self-paced, browser-based modules that students can open and use individually.

## What this repo includes

The materials in this repository are intended to support student learning about the nervous system through interactive HTML activities. These tools can be used to explore topics such as:

- introductory neuroanatomy concepts
- neuron structure and function
- synaptic transmission
- spinal cord and spinal nerve organization
- nerve study and review activities

## How to run the activities

Each activity in this repository can be opened directly in a web browser using the GitHub Pages site for this repo:

`https://docdochoggarth.github.io/Neuro/`

To run a specific activity, append the file name to the end of that URL.

### Examples

- `https://docdochoggarth.github.io/Neuro/LE-nerve-study-tool.html`
- `https://docdochoggarth.github.io/Neuro/Neuroanatomy-Intro.html`

## Notes

- These activities are intended to run directly in a browser.
- For the best experience, use the GitHub Pages URL rather than opening files locally unless you are specifically testing them in a development workflow.
- If additional activities are added to the repository, they can be launched in the same way by appending the file name to the base URL.
